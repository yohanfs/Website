#+title: Databases

* Dasar MariaDB

MariaDB adalah sistem database dan server database. Untuk mengakses
database, diperlukan /client program/ misalnya php script atau mysql
yang telah tersedia di MariaDB.

*Enter MariaDB*

#+BEGIN_EXAMPLE
    $ mysql -u root -p -h localhost
#+END_EXAMPLE
