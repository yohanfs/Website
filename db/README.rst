Dasar MariaDB 
---------------------------------------------------------------------------------

MariaDB adalah sistem database dan server database. Untuk mengakses database,
diperlukan *client program* misalnya php script atau mysql yang telah tersedia
di MariaDB. 

**Enter MariaDB**

::

	$ mysql -u root -p -h localhost
